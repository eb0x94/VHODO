Приложението е нещо като соц.мрежа за живущите във входа,който могат да сигнализират за проблеми и нередности.
То ima Navigation Drawer,в който има секции за контакт със съседи и механици(водопроводчик,жичкаджия,строител и т.н.).Има секция за 
докладване на проблем,в който потребителя има възможност да опише дадената нередност и също така по желание,може да го заснеме <todo> и да го изпрати
към централният екран на приложението.</todo>

Github link: https://github.com/eb0x94/VHODO

Contributors: eb0x94 и viliom1

The application is something like social network for neighbours in the apartment building,where they can signalise about problems and issues.
It has Navigation Drawer, which has sections for connecting with other neighbours and mechanics(electrician,plumber and etc.).It has a section
about reporting a problem(Report a problem), where the user has opportunity to describe a certain problem and if s/he has desire,s/he can
<todo>snap it with the camera and send it on the main board of the application.</todo>
